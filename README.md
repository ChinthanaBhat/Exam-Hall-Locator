\# Exam Hall Locator 🎓

A Java-based web application that dynamically allocates exam halls to students.



\## 🚀 Features

\- \*\*Dynamic Allocation\*\*: Automatically assigns students to halls based on capacity.

\- \*\*Object-Oriented Design\*\*: Uses custom `Student` and `Room` classes to manage data.

\- \*\*CSV Integration\*\*: Loads student records directly from a `students.csv` file.

\- \*\*Glassmorphism UI\*\*: A clean, modern interface with a Lavender and Amethyst theme.



\## 🛠️ Technical Stack

\- \*\*Backend\*\*: Jakarta EE Servlets

\- \*\*Frontend\*\*: JSP, HTML, CSS

\- \*\*Server\*\*: Apache Tomcat 11

\- \*\*Data\*\*: Java Collections (HashMap)



\## 📂 Project Structure

\- `src/`: Java source files (Servlet, Room, Student)

\- `web/`: JSP files and CSS

\- `WEB-INF/`: Configuration and CSV database
